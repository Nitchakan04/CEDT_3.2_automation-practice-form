# Student Registration Form - Playwright

This project contains automated test scripts for the **Student Registration Form** using **Playwright + TypeScript**.

---

# How to Install

Install dependencies

```
npm install
```

Install Playwright browsers

```
npx playwright install
```

---



# How to Run Tests

Run all tests

```
npx playwright test
```

Run specific test file

```
npx playwright test tests/registration.submit.spec.ts
```

```
npx playwright test tests/registration.validation.spec.ts
```

```
npx playwright test tests/registration.widget-ui.spec.ts
```

Run tests with browser UI

```
npx playwright test --headed
```
